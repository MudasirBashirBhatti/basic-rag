# AI Resume Assistant (RAG System)

An AI-powered Resume Chatbot built using **Retrieval-Augmented Generation (RAG)**.  
It allows users to upload a PDF resume and ask intelligent questions about its content.

---

## Features

- Load PDF resumes using PyPDFLoader
- Smart text chunking (Recursive Character Text Splitter)
- Semantic search using HuggingFace embeddings
- Vector storage with ChromaDB
- AI responses using Groq LLM (Llama 3)
- Context-aware question answering (RAG pipeline)
- Fast and lightweight architecture
